# 虫妈LTR自然拼读游戏

一个基于React开发的英语自然拼读练习工具，帮助儿童学习CVC单词（辅音-元音-辅音结构单词）。

## 功能特点

### 🎮 两种练习模式
- **练习模式**：自由拼词，无压力学习
- **闯关模式**：按关卡挑战，循序渐进

### 📚 丰富内容
- 包含100+个CVC单词
- 10个难度关卡
- 每个字母和音标都有标准发音

### 🔊 语音功能
- 点击字母播放字母发音
- 点击音标播放音标发音
- 点击单词播放完整单词发音

### 🎯 互动设计
- 拖拽字母拼词
- 点击字母槽删除字母
- 实时反馈和纠正

## 使用方法

### 练习模式
1. 点击"练习模式"进入
2. 点击字母或拖拽到拼词槽
3. 拼出单词后点击"听发音"验证
4. 点击"清空"重新开始

### 闯关模式
1. 点击"闯关模式"进入
2. 系统会显示目标单词
3. 拼出正确单词自动验证
4. 使用"←"和"→"切换单词
5. 选择不同关卡调整难度

## 技术实现

- **前端框架**：React + TailwindCSS
- **响应式设计**：适配各种屏幕尺寸
- **语音播放**：调用在线语音资源
- **交互体验**：拖拽、点击等多种操作方式

## 使用说明

1. 确保网络连接正常（用于语音播放）

## 学习建议

1. **初学者**：从第1关开始，熟练掌握基础字母发音
2. **进阶者**：尝试更高关卡，挑战复杂单词
3. **日常练习**：每天10-15分钟，持续练习效果更佳
4. **发音重点**：多听多模仿，注意短音的准确发音
